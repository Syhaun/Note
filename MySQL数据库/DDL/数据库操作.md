#查询
##查询所有数据库
`SHOW DATABASES;`
##查询当前数据库
`SELECT DATABASE();`
#创建
`CREATE DATABASE [IF NOT EXISTS] 数据库名 [DEFAULT CHARSET 字符集] [COLLATE 排列规则]`
eg:
`create database ha default charset utf8mb4;`

`create database if not exists ha default charset utf8mb4;`
#删除
`DROP DATABASE[IF EXISTS] 数据库名;`
#使用
`USE 数据库名;`
