# 使用泛型的必要性

==泛型可以在编译的时候就可以检查出错误,而不是在运行时检查出错误==

**泛型类型必须是应用类型,不能使用int double 等基本类型来代替泛型,可以使用包装类**

# 定义泛型类和接口

**可以为类或者接口定义泛型.当使用类来创建对象的时候,或者使用类或接口来声明引用变量时,必须指定具体的类型.**

# 泛型方法

**要声明泛型方法,将泛型类型<E>置于方法头中紧跟在关键字static之后**

eg:
```java
public static <E> void print(E[] list)
```
**要调用泛型方法,需要将实际类型放在尖括号内作为方法名的前缀**
