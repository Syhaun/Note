ArraysList对象可以用于存储一个对象列表

ArrayList<String> cities = new ArrayList<String>();

add(e)  

增加一个元素e到末尾哦

add(index,e)

增加一个元素e到指定下标

clear()

删除列表中的所有元素

contains(o)

判断列表中是否有元素o

get(index)

返回指定下标的元素

indexOf(o)

返回列表第一个匹配元素的下标

isEmpty()

判断列表是否为空

size()

remove(index)

去除指定下标的元素，如果该元素被去除，返回被去除的元素

set(index,e)

设置指定下标位置的元素
